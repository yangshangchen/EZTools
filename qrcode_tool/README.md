# 二维码工具箱

本地二维码生成 / 批量生成 / 解析桌面工具，基于 PyQt5。

## 功能

- **生成二维码**：输入文本或链接，自定义前景色、背景色、容错率、尺寸，支持叠加 Logo
- **批量生成**：每行一个链接，批量生成二维码并打包为 ZIP
- **解析二维码**：上传含二维码的图片，解码显示内容
- **全部本地操作**：无网络请求，数据不上传

## 直接运行（无需 Python）

双击 `二维码工具箱.exe` 即可运行。
（文件约 98MB，因内置了 Python 和所有依赖库）

## 从源码运行

```bash
cd E:\赚钱小工具\qrcode_tool
pip install -r requirements.txt
python main.py
```

## 使用说明

| 功能 | 操作 |
|------|------|
| 生成单张 | 左侧输入文本 → 调参数 → 点「生成二维码」→ 下载 PNG |
| 叠加 Logo | 点「选择 Logo」上传图片（自动 ≤100×100） |
| 批量生成 | 左侧下方多行输入 → 点「批量生成并打包 ZIP」 |
| 解析二维码 | 右侧「选择图片」→ 点「解码」 |
| 总在最前 | 勾选窗口顶部「总在最前」 |
