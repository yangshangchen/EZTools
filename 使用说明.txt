╔══════════════════════════════════════════╗
║     ⚡ 效率工具箱 EZTools — 使用指南       ║
╚══════════════════════════════════════════╝

📁 工具包位置：工具包\ 目录

所有工具均为绿色 EXE，直接双击运行即可。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. CSV数据清洗工具.exe  — 数据清洗与格式转换
   打开 → 选择CSV/Excel → 勾选操作 → 执行清洗 → 导出

2. LightningSearch_pro.exe  — 全盘文件搜索
   输入关键词 → 选择分类 → 搜索 → 双击打开或批量操作

3. LightningSearch_free.exe  — 免费文件搜索（最多100条）
   同上，适合日常快速找文件

4. PDF_Tool.exe  — PDF 合并/拆分/提取
   切换到对应标签 → 添加文件 → 设置参数 → 执行

5. 图片批量压缩工具.exe  — 图片压缩
   添加图片 → 调节质量滑块 → 开始压缩 → 导出ZIP

6. 批量重命名工具.exe  — 文件批量重命名
   添加文件 → 设置规则(前缀/序号/替换) → 预览 → 执行

7. 文件夹同步备份工具.exe  — 文件夹同步
   选择源和目标 → 选择模式 → 模拟 → 执行

8. 视频截图工具.exe  — 视频帧截图
   打开视频 → 拖拽/播放到目标帧 → 加水印 → 截图保存

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
